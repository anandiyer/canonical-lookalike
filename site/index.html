How it works

The same pipeline a human researcher runs — turned into an algorithm.

01

Profile ingestion

Reconstruct the full profile from the open web — education, every role, the company's actual product.

02

Trait extraction

Distil 5–8 defining dimensions — career arc, domain pivots, rare skill combos — each weighted by how distinctive it is.

03

Query generation

Turn the trait vector into multiple neural search queries. You can't find a lookalike with one search — you search across trait combinations.

04

Candidate retrieval

Exa neural search surfaces people by career meaning, not keywords, then results are merged and de-duplicated.

05

Scoring & ranking

Each candidate is scored on every trait axis (0–1), weighted, and ranked. The top 3–4 come back with a note on what matches — and what doesn't.
